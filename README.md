# PR Status Monitor

Displays the count of your open Pull Requests and their CI build status directly in the bottom status bar.

## Features
- **Live Counters**: Shows active open PR totals.
- **CI Status Anchors**: Integrates checkmarks `$(pass)` or error icons `$(error)` dynamically.
- **Deep Linking**: Click the status bar metric to immediately jump to the PR page on GitHub.
